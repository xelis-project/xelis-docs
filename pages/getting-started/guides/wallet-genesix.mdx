## Genesix

This is the GUI wallet for XELIS which is compatible with Android, iOS, MacOS, Windows and Linux.

You can compile or download the latest release from the [official repository](https://github.com/xelis-project/xelis-genesix-wallet).

**WARNING**: This page is a work in progress and will be updated soon.
