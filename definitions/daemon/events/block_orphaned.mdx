### `block_orphaned`

> When a block was previously ordered and is now orphaned from the main chain

On Event

```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": {
        "block_hash": "e42555732f8ca3a55bf97cbb8d63c73c0e8db8b376f60f4794871d468ffe83fc",
        "event": "block_orphaned",
        "old_topoheight": 641917
    }
}
```
