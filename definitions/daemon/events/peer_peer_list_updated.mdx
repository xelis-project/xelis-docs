### `peer_peer_list_updated`

> When a peer list of a peer has been updated.

On Event

```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": {
        "event": "peer_peer_list_updated",
        "peer_id": 7542674138406502028,
        "peerlist": [
            "1.1.1.1:2125"
        ]
    }
}
```
