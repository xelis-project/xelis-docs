### `new_block_template`

When a new mining block template is available.


On Event
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": {
        "event": "new_block_template",
        "template": "...",
        "algorithm": "xel/v2",
        "height": 106174,
        "topoheight": 107220,
        "difficulty": "85713090000"
    }
}
```
