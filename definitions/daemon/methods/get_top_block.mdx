### `get_top_block`

Retrieve the highest block based on the topological height


Parameters
|     Name    |   Type  | Required |          Note          |
|:-----------:|:-------:|:--------:|:----------------------:|
| include_txs | Boolean | Optional | Include txs serialized |

Request
```json
{
    "jsonrpc": "2.0",
    "method": "get_top_block",
    "id": 1,
    "params": {
        "include_txs": false
    }
}
```
