### `get_pruned_topoheight`

Retrieve the pruned topoheight if the node has a pruned chain.
Otherwise, returns `null` as value.


Parameters
No parameters

Request
```json
{
    "jsonrpc": "2.0",
    "method": "get_pruned_topoheight",
    "id": 1
}
```

Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": null
}
```
