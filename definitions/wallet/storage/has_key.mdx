### `has_key`

Verify if the key is present in the requested Tree.


Parameters
| Name |   Type    | Required |              Note             |
|:----:|:---------:|:--------:|:-----------------------------:|
| tree | String    | Required | Tree name to search in        |
| key  | DataValue | Required | Entry key to check            |

Request
```json
{
    "jsonrpc": "2.0",
    "method": "has_key",
    "id": 1,
    "params": {
        "tree": "test",
        "key": "my_list"
    }
}
```

Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": true
}
```
