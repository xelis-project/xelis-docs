### `decrypt_ciphertext`

> Decrypt a ciphertext from its compressed format.

Parameters

|    Name    |     Type    | Required |              Note               |
|:----------:|:-----------:|:--------:|:-------------------------------:|
| ciphertext | Compressed  | Required | Ciphertext compressed format    |

Request
```json
{
  "jsonrpc": "2.0",
  "method": "decrypt_ciphertext",
  "id": 1,
  "params": {
    "ciphertext": {
      "handle": [1, 2, 3],
      "commitment": [4, 5, 6]
    }
  }
}
```

Response
```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": 5700000000
}
```
