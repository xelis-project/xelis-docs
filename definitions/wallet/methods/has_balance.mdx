### `has_balance`

Verify if wallet has the requested asset balance.
When no parameter is set, default asset is XELIS.


Parameters
|  Name | Type | Required |     Note     |
|:-----:|:----:|:--------:|:------------:|
| asset | Hash | Optional | Asset to use |

Request
```json
{
    "jsonrpc": "2.0",
    "method": "has_balance",
    "id": 1,
    "params": {}
}
```

Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": true
}
```
