### `clear_tx_cache`

In case of a failure while broadcasting a TX from this wallet by yourself, you can erase the TX cache stored in the wallet.


Parameters
No parameters

Request
```json
{
    "jsonrpc": "2.0",
    "method": "clear_tx_cache",
    "id": 1
}
```

Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": true
}
```
