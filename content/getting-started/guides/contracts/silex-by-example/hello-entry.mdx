## Hello entry and RPC event

**Focus:** one callable `hello` entry that returns a greeting and emits one typed RPC event.

```rust filename="hello.slx" copy
struct HelloRpcEvent {
    name: string,
    message: string
}

entry hello(name: string) -> string {
    let message: string = "Hello, " + name
    fire_rpc_event(1, HelloRpcEvent { name, message })
    return message
}
```

## How it works

- `HelloRpcEvent` names the off-chain payload fields instead of sending a loose array.
- `entry hello(name: string) -> string` is the single public function in this example.
- `let message: string = "Hello, " + name` builds the value that is both returned and emitted.
- `fire_rpc_event(1, HelloRpcEvent { name, message })` creates a daemon `contract_event` notification when the entry runs on-chain.
