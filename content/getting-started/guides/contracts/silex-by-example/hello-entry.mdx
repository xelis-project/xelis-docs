## Hello entry

**Focus:** one callable `hello` entry that builds and returns a greeting.

[Open in Playground](https://playground.xelis.io/?code=ZW50cnkgaGVsbG8obmFtZTogc3RyaW5nKSAtPiBzdHJpbmcgewogICAgbGV0IG1lc3NhZ2U6IHN0cmluZyA9ICJIZWxsbywgIiArIG5hbWUKICAgIC8vIERlYnVnIHByaW50OiBieSBkZWZhdWx0LCBub2RlcyBkbyBub3Qgc2hvdyBwcmludGxuIG91dHB1dC4KICAgIHByaW50bG4obWVzc2FnZSkKICAgIHJldHVybiBtZXNzYWdlCn0%3D)

```rust filename="hello.slx" copy
entry hello(name: string) -> string {
    let message: string = "Hello, " + name
    // Debug print: by default, nodes do not show println output.
    println(message)
    return message
}
```

## How it works

- `entry hello(name: string) -> string` is the single public function in this example.
- `let message: string = "Hello, " + name` builds the returned value.
- `println(message)` is useful while debugging locally, but default node configuration does not display its output.
- `return message` produces a typed exit value. On-chain, it is stored in the contract logs.
