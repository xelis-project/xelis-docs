## Hello entry and RPC event

This is the smallest useful contract shape: one callable `entry`, a typed argument, a return value, and a `fire_rpc_event` for off-chain clients.

```rust filename="hello.slx" copy
struct HelloRpcEvent {
    name: string,
    message: string
}

entry hello(name: string) -> string {
    let message: string = "Hello, " + name
    fire_rpc_event(1, HelloRpcEvent { name, message })
    return message
}
```

`fire_rpc_event` creates a `contract_event` daemon WebSocket notification when the entry runs on-chain.
