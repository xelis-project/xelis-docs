## Introduction

This guide is a hands-on Silex tour inspired by the "by example" style. Each page introduces one idea, gives you a complete contract snippet, and links directly to Playground with the code preloaded.

<Callout type="info">
Examples that only use pure language features can be compiled and run directly in Playground. Examples that use constructors, deposits, transfers, asset creation, scheduled executions, contract events, or inter-contract calls need wallet/daemon RPC and an on-chain deployment on `devnet`.
</Callout>
